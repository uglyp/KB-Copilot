## 法律依据与合规要求

### 1. 相关法律法规

- **《中华人民共和国电子签名法》**（2004年颁布，2019年修订）
- **《电子认证服务管理办法》**（工信部）
- **《电子认证服务密码管理办法》**（国家密码管理局）

### 2. 法律效力要求

根据《电子签名法》第十四条，**可靠的电子签名与手写签名或者盖章具有同等的法律效力**。

**"可靠"的电子签名需满足以下条件：**

1. ✅ **电子签名制作数据**用于电子签名时，属于电子签名人专有
2. ✅ 签署时**电子签名制作数据**仅由电子签名人控制
3. ✅ 签署后对电子签名的任何改动能够被发现
4. ✅ 签署后对数据电文内容和形式的任何改动能够被发现

### 3. 合规性要求

#### 3.1 数字证书要求

- 必须使用**国家认可的CA机构**颁发的数字证书
- 证书必须符合**《电子认证服务管理办法》**要求
- 证书需包含：证书持有者信息、证书有效期、CA机构信息、公钥等

#### 3.2 时间戳要求

- 必须使用**国家授时中心**或**国家认可的第三方时间戳服务机构**提供的时间戳
- 时间戳需证明文件在特定时间点的存在性和完整性

#### 3.3 存证要求

- 签署过程需完整记录（操作日志、IP地址、设备信息等）
- 签署后的文件需进行**哈希值计算**并存储
- 需建立**不可篡改的存证链**

### 1. 签章服务核心逻辑（Java示例）

```java
// SealService.java
@Service
public class SealService {

    @Autowired
    private CertificateService certificateService;

    @Autowired
    private TimestampService timestampService;

    @Autowired
    private EvidenceService evidenceService;

    /**
     * 应用电子签章
     */
    public SealResult applySeal(SealRequest request) {
        // 1. 验证证书有效性
        CertificateInfo cert = certificateService.getCertificate(request.getCertificateId());
        if (!cert.isValid()) {
            throw new BusinessException("证书无效或已过期");
        }

        // 2. 计算文档哈希值
        byte[] documentBytes = loadDocument(request.getDocumentId());
        String documentHash = calculateSHA256(documentBytes);

        // 3. 使用私钥对哈希值进行签名
        String signature = signWithPrivateKey(documentHash, cert.getPrivateKey());

        // 4. 获取时间戳
        TimestampInfo timestamp = timestampService.getTimestamp(documentHash);

        // 5. 将签章信息嵌入PDF
        byte[] sealedPdf = embedSealToPdf(
            documentBytes,
            signature,
            cert,
            timestamp,
            request.getPosition()
        );

        // 6. 保存签章后的PDF
        String sealedPdfUrl = saveSealedDocument(sealedPdf, request.getDocumentId());

        // 7. 记录存证信息
        SealRecord record = SealRecord.builder()
            .documentId(request.getDocumentId())
            .documentHash(documentHash)
            .certificateId(request.getCertificateId())
            .signature(signature)
            .timestamp(timestamp.getTimestamp())
            .position(request.getPosition())
            .sealedPdfUrl(sealedPdfUrl)
            .build();

        evidenceService.saveSealRecord(record);

        return SealResult.builder()
            .sealId(record.getId())
            .sealedPdfUrl(sealedPdfUrl)
            .timestamp(timestamp.getTimestamp())
            .certificateInfo(cert)
            .build();
    }

    /**
     * 验证电子签章
     */
    public VerifyResult verifySeal(String documentId) {
        // 1. 获取签章记录
        SealRecord record = evidenceService.getSealRecord(documentId);
        if (record == null) {
            return VerifyResult.fail("未找到签章记录");
        }

        // 2. 提取PDF中的签章信息
        byte[] sealedPdf = loadDocument(record.getSealedPdfUrl());
        SealInfo sealInfo = extractSealFromPdf(sealedPdf);

        // 3. 验证文档完整性（计算哈希值）
        byte[] documentBytes = removeSealFromPdf(sealedPdf);
        String currentHash = calculateSHA256(documentBytes);
        if (!currentHash.equals(record.getDocumentHash())) {
            return VerifyResult.fail("文档已被篡改");
        }

        // 4. 验证数字签名（使用公钥验证）
        boolean signatureValid = verifySignature(
            record.getDocumentHash(),
            sealInfo.getSignature(),
            sealInfo.getPublicKey()
        );
        if (!signatureValid) {
            return VerifyResult.fail("数字签名验证失败");
        }

        // 5. 验证时间戳
        boolean timestampValid = timestampService.verifyTimestamp(
            record.getDocumentHash(),
            record.getTimestamp()
        );
        if (!timestampValid) {
            return VerifyResult.fail("时间戳验证失败");
        }

        // 6. 验证证书有效性
        CertificateInfo cert = certificateService.getCertificate(record.getCertificateId());
        if (!cert.isValid()) {
            return VerifyResult.fail("证书已过期或无效");
        }

        return VerifyResult.success(sealInfo, cert, record.getTimestamp());
    }

    /**
     * 使用私钥签名
     */
    private String signWithPrivateKey(String data, PrivateKey privateKey) {
        try {
            Signature signature = Signature.getInstance("SHA256withRSA");
            signature.initSign(privateKey);
            signature.update(data.getBytes(StandardCharsets.UTF_8));
            byte[] signedBytes = signature.sign();
            return Base64.getEncoder().encodeToString(signedBytes);
        } catch (Exception e) {
            throw new BusinessException("签名失败", e);
        }
    }

    /**
     * 验证签名
     */
    private boolean verifySignature(String data, String signature, PublicKey publicKey) {
        try {
            Signature sig = Signature.getInstance("SHA256withRSA");
            sig.initVerify(publicKey);
            sig.update(data.getBytes(StandardCharsets.UTF_8));
            byte[] signatureBytes = Base64.getDecoder().decode(signature);
            return sig.verify(signatureBytes);
        } catch (Exception e) {
            return false;
        }
    }

    /**
     * 计算SHA-256哈希值
     */
    private String calculateSHA256(byte[] data) {
        try {
            MessageDigest digest = MessageDigest.getInstance("SHA-256");
            byte[] hash = digest.digest(data);
            return Hex.encodeHexString(hash);
        } catch (Exception e) {
            throw new BusinessException("哈希计算失败", e);
        }
    }
}
```

    /**
     * 将签章信息嵌入PDF
     */

        byte[] pdfBytes,
        String signature,
        CertificateInfo cert,
        TimestampInfo timestamp,
        SealPosition position
    ) throws Exception {
        ByteArrayOutputStream outputStream = new ByteArrayOutputStream();

        PdfReader reader = new PdfReader(pdfBytes);
        PdfSigner signer = new PdfSigner(reader, outputStream, new StampingProperties());

        // 设置签章外观
        PdfSignatureAppearance appearance = signer.getSignatureAppearance();
        appearance.setReason("电子签章");
        appearance.setLocation("中国");
        appearance.setPageNumber(position.getPage());
        appearance.setPageRect(new Rectangle(
            position.getX(),
            position.getY(),
            position.getX() + 200,
            position.getY() + 100
        ));

        // 创建签章外观（包含证书信息）
        Image sealImage = createSealImage(cert, timestamp);

## 签章验证流程

### 完整验证流程

```
1. 用户上传已签章PDF
   ↓
2. 提取PDF中的签章信息
   - 数字签名
   - 证书信息
   - 时间戳
   ↓
3. 验证文档完整性
   - 计算当前文档哈希值
   - 与签章时的哈希值对比
   ↓
4. 验证数字签名
   - 使用证书公钥验证签名
   ↓
5. 验证证书有效性
   - 检查证书是否过期
   - 调用CA机构OCSP服务验证证书状态
   ↓
6. 验证时间戳
   - 验证时间戳令牌的有效性
   ↓
7. 返回验证结果
   - 验证通过：显示证书信息、时间戳等
   - 验证失败：显示失败原因
```

### ⚠️ 合规性要求

1. **必须使用国家认可的CA机构**颁发的数字证书
2. **必须使用国家认可的时间戳服务**
3. **必须完整记录签章过程**（操作日志、IP、设备信息等）
4. **必须保证签章数据的不可篡改性**
5. **建议使用区块链存证**增强法律效力

### 🔒 安全性要求

1. **私钥安全**：私钥必须安全存储，不能泄露
2. **传输加密**：所有API调用必须使用HTTPS
3. **访问控制**：严格控制证书和签章的访问权限
4. **审计日志**：完整记录所有签章操作

### 📝 最佳实践

1. **证书管理**：建立完善的证书申请、更新、吊销流程
2. **存证备份**：签章数据需多重备份，防止数据丢失
3. **定期验证**：定期验证已签章文档的有效性
4. **用户教育**：向用户说明电子签章的法律效力和使用规范
